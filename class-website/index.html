탁월한 기능.
놀라운 단순함.
복잡한 설정 없이, 설치하면 바로 씁니다. 필요한 건 폴더 경로 두 개뿐.
01
⚡
즉각 분류
키워드 매칭 방식으로 AI 응답 대기 없이 파일이 생기는 즉시 분류됩니다.
02
👁
실시간 감시
다운로드 폴더를 감시하다가 새 파일이 생기면 자동으로 올바른 위치로 이동합니다.
03
📦
일괄 분류
수백 개 파일이 쌓인 폴더도 "지금 분류 실행" 한 번으로 몇 초 만에 정리됩니다.
04
🕐
저장 이력
모든 저장 파일의 전체 경로와 시각을 기록합니다. 더블클릭으로 탐색기에서 바로 열기.
05
🖱
드래그 이동
이력 목록에서 파일을 끌어 폴더 트리에 놓으면 즉시 이동됩니다.
06
🔒
완전 오프라인
인터넷 연결이 필요 없습니다. 데이터는 내 PC 밖으로 나가지 않습니다.
사용법

4단계면 충분합니다.

01
설치
Class_Setup.exe를 실행해 설치합니다. 언어와 바탕화면 바로가기를 선택하세요.
02
폴더 설정
소스 폴더(파일이 쌓이는 곳)와 대상 폴더(분류 결과가 저장될 곳)를 지정합니다.
03
감시 시작
실시간 감시를 켜거나 "지금 분류 실행"을 클릭하세요. 창을 닫아도 백그라운드에서 계속 동작합니다.
04
자동 정리
파일명과 확장자를 분석해 가장 적합한 하위 폴더로 자동 이동됩니다.
